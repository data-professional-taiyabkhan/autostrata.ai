## The Problem

Sartorial London operates a premium designer fashion rental marketplace built on Sharetribe Flex. The platform needed significant custom engineering beyond the default Sharetribe template — a fixed 5-day rental engine, a better mobile browsing experience, a dynamic product discovery component, and groundwork for Stripe card holds.

Their team needed a fast turnaround — custom marketplace features shipped in weeks, not months.

## Our Solution

### Fixed 5-Day Rental Engine
We built a renter-side calendar locked to a 5-day window with auto-calculated end dates and full-window availability validation. The booking flow prevents overlapping rentals and handles edge cases around partial availability.

### Custom Featured Fits Carousel
A dynamic component pulling the 5–6 most recent listings via the Sharetribe API, with a CMS section-name override for editorial control. This gave shoppers outfit inspiration while reducing friction to purchase.

### Mobile Category Grid
Replaced the default vertical slider with a Gucci-inspired 2×2 grid using Tailwind responsive utilities. The result is a faster, more visual browsing experience on mobile devices.

### Shipping Fees for Rentals
Extended the pricing logic to add shipping line-items to rental transactions, not just sale items. This required changes to the transaction process at the Sharetribe Console level.

### Stripe Deposit Prep
Console-level documentation and design for card pre-authorisation with auto-capture before the 7-day Stripe limit. This groundwork enables the client to add damage deposits in a future sprint.

## Tech Stack

- **Platform:** Sharetribe Flex
- **Frontend:** React, Tailwind CSS
- **Payments:** Stripe Connect
- **Integrations:** Sharetribe API, Mailchimp, Google Analytics 4
- **Hosting:** Vercel
